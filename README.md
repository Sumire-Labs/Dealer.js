## 📖 概要

Dealer.js は、GTAの世界観をテキストベースの Discord BOT として再現したゲームです。カジノでギャンブルに興じたり、労働で地道に稼いだり、あるいは強盗で一攫千金を狙ったり——プレイヤーそれぞれのスタイルで遊ぶことができます。

## 🎮 ゲームコンテンツ

### カジノ

| ゲーム | 説明 |
|---|---|
| 🎰 スロット | リールを回して絵柄を揃えろ |
| 🃏 ブラックジャック | ディーラーとの 21 勝負 |
| 🪙 コインフリップ | 表か裏か——シンプルな賭け |
| 🎯 ルーレット | 数字・色・範囲を予想して賭ける |
| 🃏 ポーカー | 手札の強さで勝負を決めろ |

### アクティビティ

| コンテンツ | 説明 |
|---|---|
| 🏇 競馬 | レースの勝ち馬を予想して賭ける |
| 💼 労働 | 真面目に働いて安定した収入を得る |
| 🔫 強盗 | ハイリスク・ハイリターンの犯罪行為 |
| 🔒 刑務所 | 捕まったら服役——しばらくお預け |
| 🏃 脱獄 | 刑務所からの脱出を試みる |
