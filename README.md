# Dealer.js

Discord カジノ Bot。スロット、ブラックジャック、コインフリップ、競馬、ポーカーなどのギャンブルゲームと、銀行・ローンシステムを搭載。

## 主な機能

- **5 種類のカジノゲーム** — スロット / ブラックジャック / コインフリップ / 競馬 / ポーカー
- **経済システム** — チップ残高、デイリーボーナス、リーダーボード
- **銀行システム** — 口座預入・出金・プレイヤー間送金・日次利息
- **ローンシステム** — 借入・返済・自己破産
- **管理者コマンド** — チップ付与、ユーザーリセット、Bot 設定
